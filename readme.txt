# 📝 나의 투두리스트 웹앱

웹 기반 투두리스트 애플리케이션입니다.
카테고리별 할일 관리, 리스트 추가/삭제, 로컬 저장 및 불러오기 기능을 지원하며, PC와 모바일 환경에서 모두 최적화되어 있습니다.

---

## 🌟 주요 기능

### 1️⃣ 리스트 관리

* **새 리스트 생성**: "+ 새 리스트 생성" 버튼 클릭
* **리스트 삭제**: 리스트 상단 `X` 버튼 클릭 후 확인
* **리스트 제목 수정**: 리스트 제목 옆 `...` 버튼 클릭 후 편집

### 2️⃣ 할일(Todo) 관리

* **할일 추가**: 리스트 내 "+ 새 할일" 버튼 클릭
* **할일 삭제 대기**: 할일 오른쪽 `X` 버튼 클릭 → 붉은색 표시 후 저장 시 삭제
* **내용 수정**: 할일 내용 클릭 후 수정 (최대 16자)
* **완료 체크**: 체크박스 클릭으로 완료 상태 표시
* **카테고리 변경**: 드롭다운에서 카테고리 선택 가능

### 3️⃣ 카테고리 관리

* **새 카테고리 추가**: 상단 입력창 → "추가" 버튼 클릭
* **카테고리 수정/삭제**: 카테고리 클릭 → 수정/삭제 버튼 사용
* **카테고리 순서 이동**: 수정 모드에서 `←`, `→` 버튼
* **카테고리 필터**: 상단에서 선택 시 해당 항목만 표시

### 4️⃣ 데이터 저장 및 불러오기

* **로컬 저장**: "로컬 저장" 버튼 클릭 → JSON 파일 다운로드
* **파일 불러오기**: "불러오기" 버튼 클릭 → JSON 선택
* **자동 로컬 저장**: 리스트 저장 버튼 클릭 시 반영

### 5️⃣ 정렬 기능

* **최신순**: 생성일 기준 내림차순
* **입력순**: 생성일 기준 오름차순
* **수행순**: 완료되지 않은 할일이 많은 순

### 6️⃣ 스크롤

* **리스트 가로 스크롤**: 마우스 휠, 터치 드래그
* **카테고리 가로 스크롤**: 마우스 휠 좌우 이동

---

## 🚀 사용 방법

1. 웹 브라우저에서 `index.html` 열기
2. 리스트와 할일 생성 및 관리
3. 변경 사항 저장: 리스트 하단 "저장" 버튼 클릭
4. 카테고리 관리로 할일 분류 및 필터링
5. 데이터 백업: "로컬 저장" → 복원 시 "불러오기"

---

## 💻 기술 스택

* **HTML5**
* **CSS3** (반응형, Flex/Grid 사용)
* **JavaScript** (Vanilla JS, LocalStorage 활용)

---

## ⚠️ 주의 사항

* 로컬 스토리지를 사용하므로, 다른 컴퓨터에서는 저장 데이터 확인 불가
* 저장하지 않은 변경 사항은 새로고침 시 사라짐
* 할일 내용 최대 16자 입력 가능
